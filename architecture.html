⑥ Security Design
🚫
No Direct AWS Calls
React never calls AWS SNS, DynamoDB, or any AWS SDK. All AWS interactions happen exclusively in the backend Lambda.
🔑
JWT via Bearer Token
Every protected API request includes Authorization: Bearer <token> injected by the Axios request interceptor.
💾
sessionStorage Scope
JWT lives in sessionStorage — automatically cleared when the browser tab closes. No localStorage persistence reduces attack surface.
⏱️
OTP Expiry + Rate Limit
60-second resend timer in UI prevents spam. Backend enforces TTL and rate limiting on OTP generation independently.
🔄
Auto 401 Logout
Axios response interceptor detects 401 Unauthorized, clears sessionStorage, and redirects to login — no manual handling needed per page.
🌐
HTTPS + CORS
All API calls are made over HTTPS. CORS is enforced on the API Gateway, ensuring only your Amplify domain can call the backend.
